# Tax-Insight | 个人税务与财务情报看板

一个专业的个人税务情报平台，提供最新税务新闻展示和个人所得税计算功能。

## 功能特性

### 1. 个人所得税计算器
- ✅ 支持 2024-2026 现行税率标准
- ✅ 7 级超额累进税率（3%-45%）
- ✅ 专项附加扣除（子女教育、住房贷款利息、赡养老人）
- ✅ 实时计算税后收入和年度预估
- ✅ 直观的结果展示

### 2. 税务新闻资讯
- ✅ 卡片瀑布流展示
- ✅ 包含新闻标题、摘要、配图
- ✅ 标签分类（政策变动、税务解读、通知公告等）
- ✅ 响应式设计，适配移动端

### 3. 专业商务风格
- ✅ 深色/浅色主题切换
- ✅ 简洁的卡片式布局
- ✅ 蓝色系专业配色
- ✅ 流畅的交互动效

## 技术栈

- **框架**: Next.js 15 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **主题**: next-themes
- **图标**: Lucide React
- **数据**: Mock 本地数据

## 主要功能说明

### 个税计算

计算器基于 2024-2026 年个税政策：
- 起征点：5000 元/月
- 支持五险一金扣除
- 支持 3 种常用专项附加扣除
- 自动计算月度税额和年度预估

### 专项附加扣除标准

- 子女教育：2000 元/月
- 住房贷款利息：1000 元/月
- 赡养老人：3000 元/月
